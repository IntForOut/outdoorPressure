# OutdoorPressure Knowledge Graph

This knowledge graph supports the study of the pressure of human outdoor leisure on moutain ecosystems. It embeds concepts relevant to express questions about the ecological pression caused by human activities on wildlife in mountains, and a catalogue of different assets relevant to investigate these questions, like datasets, reproducible processes, scientific papers, or pedagogic materials. Documentation: **[OutdoorPressure ontology](https://intforout.github.io/outdoorPressure/index.html)**
